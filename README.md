# Avito Internship Test Task Repository

## QA

### Вопрос 1. Может ли пользователь быть привязанным к нескольким командам? 

> Привязываем пользователя только к одной команде, так как бэкендеру нет смысла проводить ревью ml-инженеру или фронтендеру

### Вопрос 2. Как корректно реализовать изменение команды? 

> Если получаем запрос на /team/add с существующей командой и новыми юзерами, добавляем их в команду, 
> Если хотя бы 1 пользователь есть в команде то кидаем ошибку
> Если пользователь(id) уже привязан к другой команде, то кидаем ошибку

## Запуск 

1. Создайте файл .env на основе .env.example 

2. Прогнать тесты 
```bash 
docker compose run --rm tests
docker compose stop postgres
docker compose down -v --remove-orphans
```

3. Запустить сервис
```bash 
docker compose up -d 
```
